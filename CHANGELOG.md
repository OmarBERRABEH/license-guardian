# Changelog

All notable changes to this project will be documented in this file. See [standard-version](https://github.com/conventional-changelog/standard-version) for commit guidelines.

## 1.0.0 (2025-11-15)

### Features

* **core**: Initial release of @license-guardian/licenses
* **checker**: License compliance checker for commercial use
* **cli**: Command-line interface with verbose, JSON, and quiet modes
* **config**: Configurable license allowlist via .licenserc.json
* **monorepo**: Automatic workspace detection and scanning

### Documentation

* **readme**: Comprehensive API documentation
* **examples**: CI/CD integration examples for GitHub Actions, GitLab CI, and CircleCI
